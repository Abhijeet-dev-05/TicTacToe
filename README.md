# ❌⭕ Tic Tac Toe Game

A fun and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript.

## 🚀 Live Demo

👉 https://tic-tac-toe-beryl-ten-85.vercel.app/

## 📌 About the Project

This project is a classic Tic Tac Toe game where two players can play against each other. It features a clean UI and smooth gameplay experience. The project demonstrates DOM manipulation and game logic implementation using JavaScript.

## 💡 Features

* 👥 Two-player mode
* 🎯 Win detection logic
* 🔄 Game reset functionality
* 🎨 Clean and simple UI
* 📱 Responsive design

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
